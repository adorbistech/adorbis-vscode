# Adorbis AI for VS Code

**One key. Every model. Auto-routed.**

Connect VS Code, Cline, Continue, Claude Code, Aider, Roo Code, and OpenCode to the Adorbis AI control plane in 30 seconds.

## Setup

1. Run `Adorbis: Setup` from the Command Palette (Ctrl+Shift+P)
2. Paste your Adorbis API key and click Connect
3. Click your AI tool to auto-write its config
4. Done — your credit balance appears in the status bar

Get a free key at [ai.adorbistech.com](https://ai.adorbistech.com)

## Commands

- `Adorbis: Setup` — Connect your API key
- `Adorbis: Check Credit Balance` — Refresh balance
- `Adorbis: Configure AI Tool` — Write config for a specific tool
- `Adorbis: Open Dashboard` — Opens ai.adorbistech.com/dashboard
- `Adorbis: Sign Out` — Remove your API key
